# FootPedalDJ
_my first hardware project_

A MIDI device to help in DJ'ing as well as a foot activated keyboard.

Useing [this](https://www.youtube.com/watch?v=H3WevrsmO9o) video as a tutorial I can make a MIDI foot pedel. but i want more, and need to change it for blueprint!

By adding LED lights to the side and making so the farmost right button changes the profile your in, keyboard or MIDI we have a changed project!

### Tools Needed:

- Soldering Iron
- Drill with 1/2" Stepped Drill Bit
- Automatic Center Punch

### How to assemble

- Follow [this](https://www.youtube.com/watch?v=H3WevrsmO9o) guide
- Connect LED's
- Close up & Power
